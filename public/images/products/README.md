# 📸 產品圖片目錄

## 文件命名規範

圖片文件名必須與產品ID完全匹配：

```
labubu-001.jpg  -> 對應產品ID "labubu-001"
labubu-002.png  -> 對應產品ID "labubu-002"
labubu-003.webp -> 對應產品ID "labubu-003"
```

## 支持的圖片格式

- `.jpg` / `.jpeg` (推薦)
- `.png` 
- `.webp` (最佳性能)
- `.gif` (動圖)

## 建議尺寸

- **最小尺寸**：400x400 像素
- **推薦尺寸**：800x800 像素 
- **最大尺寸**：1200x1200 像素
- **長寬比**：1:1 (正方形)

## 文件大小

- 單張圖片建議不超過 500KB
- 使用圖片壓縮工具優化文件大小

## 使用步驟

1. **準備圖片**：重命名為對應的產品ID
2. **放入此目錄**：`/public/images/products/`
3. **更新JSON**：修改產品數據中的 `imageUrl`

### 更新JSON範例

```json
{
  "id": "labubu-001",
  "imageUrl": "/images/products/labubu-001.jpg"
}
```
